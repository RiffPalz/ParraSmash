# 🏸 ParraSmash

**ParraSmash** is a modern, professional web application designed for seamless badminton court reservations. Built with a focus on elegant UI/UX and mobile-first responsiveness, it provides players and administrators with a streamlined booking experience.

---

## ✨ Features

* **Dynamic Booking Grid:** An intuitive interface for selecting available time slots and courts.
* **Single-Page Architecture:** Fast, fluid navigation powered by React.js.
* **Professional UI:** A clean, business-oriented aesthetic utilizing a modern color palette and typography.
* **Mobile-First Design:** Fully optimized for on-the-go bookings, ensuring the header, grid, and forms adapt perfectly to any screen size.
* **Feature Highlights:** Dedicated sections for facility amenities, membership perks, and contact information.

## 🛠️ Tech Stack

* **Frontend:** React.js (Vite)
* **Styling:** Tailwind CSS (Focusing on a clean, professional look)
* **Icons/Assets:** Lucide React / Heroicons
* **Deployment:** Optimized for Vercel / Netlify
